This is a containerized full stack MEAN application

What is what

  • A Dockerfile is a recipe for creating Docker images
  • A Docker image gets built by running a Docker command (which uses that Dockerfile)
  • A Docker container is a running instance of a Docker image

How it works

The project will have more than 1 Dockerfile

Each file will be used to get the final and optzmized Docker images. NOte that in the Angular Dockerfile, multiple stages will be decalred, one for building the project and the other for serving it through either Apache or Nginx

Docker compose will be used to build the dockerfiles and group them into 1 network. We will also use it to specify a volume to be used by the MySQL container later on so the data aren't lost

Once the images are built, we can start/stop them using either command line or Docker Desktop client.
